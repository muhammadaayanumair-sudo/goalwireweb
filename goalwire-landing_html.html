// faq

Common questions.

Is GoalWire free to use?+
Yes — GoalWire is completely free. All 88 commands are available at no cost to any Discord server.
Which leagues does GoalWire cover?+
GoalWire covers 100+ leagues and competitions including the Premier League, La Liga, Bundesliga, Serie A, Ligue 1, Champions League, Europa League, and all major international competitions including the 2026 World Cup.
How do I find my Discord Server ID?+
Open Discord Settings → Advanced → enable Developer Mode. Then right-click your server icon in the sidebar and click "Copy Server ID". Paste that 18–19 digit number into the dashboard above.
Is betting real money?+
No. All betting in GoalWire uses virtual server coins — there is no real money involved. It's purely for fun and friendly competition within your community.
Can I invite GoalWire to multiple servers?+
Absolutely. GoalWire can be added to unlimited servers. Each server has its own leaderboard, coin economy, card collections, and alert configuration.
How does the card collecting system work?+
Claim your daily coins with /claim, then open card packs with /pack to collect player cards. Trade on the /market, swap directly with /trade, and upgrade duplicates with /upgrade to earn rarer versions.
